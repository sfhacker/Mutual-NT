Pasos
-----

1.- Crear una clase concreta para una entity en particular (e.g. Paises)

2.- Dicha clase debe heredar de 'UserTemplateBase'

3.- Especificar las columnas que deberian aparecer en la grilla (e.g. GenerateGridColumns())

4.- Especificar el origen de los datos (contenido de la grilla)

5.- Especificar algun 'skin' por defecto (optional)

6.- That's all, folks!
