
Aplicacion Principal
--------------------


Descripcion
-----------

- Cada ventanita or formulario (e.g. Paises) tiene un nombre univoco

- Una o mas ventanitas se encuentran agrupadas bajo alguna categoria (e.g. Caja, Prestamo, etc)
  Esto equivale a un assembly por cada grupo (en este ejemplo, el assembly EIV.UI.UserTemplates.Tests.dll)
  
- Por cada menu item, se busca y renderiza una ventanita en particular en runtime!
